# プロジェクト概要 (PROJECT_SPEC)

## 1. プロジェクトの目的
本プロジェクトの目的は、ダイヤモンドアンビルセル（DAC）等の高圧力環境下における【NVセンターのODMR（光検出磁気共鳴）シグナルを高精度に測定・自動解析するシステム】をPythonで開発することである。

## 2. 達成すべきゴール
- 測定の完全自動化：手動操作を減らし、長時間の安定した測定（CWスキャン・FMスキャン）を実現する。
- リアルタイム・トラッキング：温度変化や磁場変動によるピークシフトに自動で追従する。
- 即時解析：取得したデータから自動でLorentzianフィッティングを行い、ZFS（ゼロ磁場分裂パラメータ D）からリアルタイムに圧力（GPa）を算出する。
- 安全性の確保：実験装置（DAQ、マイクロ波ジェネレータ）への過負荷や、接続の異常終了を防ぐ。

## 3. 必須要件
- 機器制御モジュールと解析モジュールは疎結合に保つこと。
- 設定（COMポート、スキャン範囲、トラッキングの閾値など）はすべて外部の `config.json` から読み込むこと。
- 論文（thesisフォルダ内）や最新の物理学の知見に基づいた、妥当性のあるコードを生成すること。

## 4. 禁止事項
- ハードウェアが接続されっぱなし（フリーズ状態）になるような不安定な処理。
- 物理的な単位や意味を無視したマジックナンバーの使用。
- 大規模な変更を一度に一気に適用すること（必ず小さなステップで進める）。
